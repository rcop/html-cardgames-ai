Regras Básicas do Jogo de Dados

  • O objetivo é acumular a maior pontuação possível. O jogo termina quando não houver mais combinações para pontuar.
  • Você começa cada rodada com **5 dados** e terá **3 rolagens** por rodada.
  • **A rodada inicia com sua primeira rolagem.** Após a primeira rolagem, você pode **prender** qualquer dado clicando nele (ele ficará amarelo). Dados presos não serão relançados. Clicar novamente em um dado preso o **libera**.
  • Após qualquer rolagem, você pode **escolher uma combinação na tabela ao lado para pontuar**, clicando no botão "Pontuar: X" ao lado dela. A pontuação será calculada com base nos **dados atualmente na mesa**, independentemente de estarem presos ou não.
  • **Uma vez que uma combinação é pontuada, ela não pode ser usada novamente** na partida. Ela será marcada na tabela. Você pode pontuar 0 pontos.
  • Se você não pontuar após 3 rolagens, ou se em qualquer momento os dados rolados não permitirem **nenhuma combinação disponível para pontuar**, você será obrigado a **escolher uma combinação disponível na tabela para ELIMINAR**, sem pontuar.
  • A cada rodada, ao pontuar (ou eliminar), seus dados são resetados para a próxima rodada e uma nova rodada começa.

Detalhes da Pontuação:

  • **Quinteto (5 dados iguais):** 50 pontos.
  • **Sequência Alta (2-3-4-5-6):** 40 pontos.
  • **Sequência Baixa (1-2-3-4-5):** 30 pontos.
  • **Full-House (um trio e um par):** 25 pontos.
  • **Quadra (4 dados iguais):** Soma dos 4 dados iguais + valor do quinto dado.
  • **Trinca (3 dados iguais):** Soma dos 3 dados iguais.
  • **Quantidade de X (X=1 a 6):** Soma dos dados que mostram o número X. (Ex: Três 3s (3-3-3-4-5) pontuado como Quantidade de 3s vale 3+3+3 = 9 pontos).
  • **Aleatória (qualquer combinação):** Soma de todos os 5 dados.
