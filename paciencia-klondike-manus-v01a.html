Como Jogar Paciência Klondike

Objetivo: Mover todas as cartas para as quatro fundações, organizadas por naipe em ordem crescente (Ás ao Rei).

Regras Básicas:

  • Tableau: Organize cartas em ordem decrescente alternando cores (vermelho sobre preto, preto sobre vermelho).
  • Fundações: Construa sequências do Ás ao Rei por naipe (♠ ♥ ♦ ♣).
  • Maço de Compra: Clique para revelar novas cartas. Quando vazio, clique novamente para reciclar.
  • Movimentação: Arraste cartas ou clique para selecionar e depois clique no destino.
  • Duplo Clique: Move automaticamente cartas elegíveis para as fundações.

Sistema de Pontuação

AçãoPontos

Mover carta do Tableau para Fundação+10
Virar carta no Tableau+5
Mover carta da Fundação para Tableau-15
Reciclar Maço de Compra-100

Controles

  • Arrastar e Soltar: Clique e arraste cartas para movê-las.
  • Clique Simples: Selecione uma carta e clique no destino.
  • Duplo Clique: Move automaticamente para fundação se possível.
  • Desfazer: Use o botão "Desfazer" para voltar o último movimento.
